🚀 Follow these steps to deploy UltraBot on Render for free (or paid for 24/7 uptime).
📋 Step-by-Step: Deploy on Render
1
Create a Discord Application & Bot
Go to discord.com/developers/applications → New Application → Name it → Go to "Bot" tab → Add Bot → Copy the Token (keep it secret!).
Enable these Intents: Presence Intent, Server Members Intent, Message Content Intent.
Under OAuth2 → URL Generator: select bot + applications.commands → Permissions: Administrator → Copy the URL and invite the bot to your server.
# You'll need this token in step 4
DISCORD_TOKEN=your_bot_token_here
ANTHROPIC_API_KEY=your_anthropic_key_here
2
Push the Files to GitHub
Create a new GitHub repository (can be private). Upload these files: bot.py, requirements.txt. Do NOT upload config.json or any .json files with tokens. Add a .gitignore:
# .gitignore
config.json
warnings.json
staff_codes.json
reports.json
premium.json
counting.json
notes.json
timers.json
*.pyc
__pycache__/
.env
3
Create a Render Account & New Service
Go to render.com → Sign up (free) → Click "New +" → Select "Background Worker" (important — NOT web service, bot has no HTTP server).
Connect your GitHub account → Select your bot repository.
4
Configure the Render Service
Fill in these settings in the Render dashboard:
Name: ultrabot
Runtime: Python 3
Build Command: pip install -r requirements.txt
Start Command: python bot.py
Instance Type: Free (or Starter $7/mo for 24/7)
5
Add Environment Variables
In Render → Your Service → Environment tab → Add these variables:
DISCORD_TOKEN = your_discord_bot_token
ANTHROPIC_API_KEY = sk-ant-api03-your_key_here
6
Deploy & First Run Setup
Click Deploy. Watch the logs — you should see ✅ UltraBot is online! and 🔄 Synced X slash commands.

In your Discord server, run these commands to finish setup:
# 1. Set yourself as owner (first time only)
/setowner

# 2. Set up channels
/setlog #logs
/setmodlog #mod-logs
/setwelcome #welcome
/setreport #reports
/setbotchannel #bot-commands
/setcounting #counting
/setstaff #staff-chat

# 3. Add staff & admin roles
/addstaffrole @Moderator
/addadminrole @Admin

# 4. Generate today's staff code (owner only)
/generatecode

# 5. Set website & dashboard links
/setwebsite https://yoursite.com
/setdashboard https://yourdashboard.com
7
⚠️ Free Tier Limitation & Fix
Render's free tier spins down after 15 mins of inactivity. For Discord bots (which are always-on background workers), this usually isn't an issue since the bot keeps the connection alive.

If your bot goes offline, upgrade to Starter ($7/mo) for guaranteed 24/7 uptime, or use UptimeRobot (free) to ping a keep-alive endpoint.
# Add this to bot.py for a keep-alive HTTP endpoint
# (needed if using Render Web Service instead of Background Worker)
from aiohttp import web
async def health(request):
    return web.Response(text="OK")
app = web.Application()
app.router.add_get('/', health)
